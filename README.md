## Feedback & Bug Reports

Our source code is proprietary, but we want to hear from you. Each product has a public feedback repository where you can:

- **Report bugs** — something broken or behaving unexpectedly
- **Request features** — ideas for improvements or new capabilities
- **Ask questions** — if something isn't clear or you need help

Just open an issue in the relevant feedback repo linked above. We read everything.

### Before opening an issue

1. **Search existing issues** to check if it's already been reported.
2. **Include your environment** — OS, GPU, RAM, and product version if applicable.
3. **Describe what happened** vs. what you expected — steps to reproduce help us fix things faster.

## General Enquiries

For anything that doesn't fit a specific product, open an issue in [this repo](https://github.com/AtheneumAI/.github/issues) or reach out at [atheneumai.co.uk](https://atheneumai.co.uk).
